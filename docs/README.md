# Catalogo Carol's Web

Este directorio contiene la version estatica del sitio lista para GitHub Pages.

## Publicacion

1. Sube el proyecto a GitHub.
2. En el repositorio, abre `Settings > Pages`.
3. En `Build and deployment`, elige `Deploy from a branch`.
4. Selecciona la rama principal y la carpeta `/docs`.
5. Guarda los cambios y GitHub publicara el sitio.

## Notas

- `index.html` equivale a `MainActivity`.
- Cada categoria tiene su propia pagina HTML.
- `agregar-prendas.html` y `editor-item.html` guardan datos en `localStorage`.
- Esta version no depende de Android ni de Firebase para funcionar en GitHub Pages.
